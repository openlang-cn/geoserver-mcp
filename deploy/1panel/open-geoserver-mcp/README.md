## 产品介绍

**GeoServer MCP** 是一款 GeoServer MCP（Model Context Protocol）服务器，将 GeoServer REST API 封装为 MCP 工具，供 Codex、Cursor、Claude Desktop 等 MCP 兼容客户端使用。支持工作区、数据存储、图层、样式、用户管理和系统运维等操作。

## 主要功能

- **GeoServer 全功能管理**：通过 MCP 工具管理工作区、数据存储、图层、图层组、样式、用户和系统配置
- **多传输协议支持**：支持 Streamable HTTP 和 SSE 两种远程传输协议，适配不同 MCP 客户端
- **一键部署**：通过 Docker 容器化部署，预配置环境变量，开箱即用
