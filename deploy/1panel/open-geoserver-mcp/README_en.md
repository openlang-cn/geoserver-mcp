## Introduction

**GeoServer MCP** is a GeoServer MCP (Model Context Protocol) server that wraps the GeoServer REST API into MCP tools for MCP-compatible clients such as Codex, Cursor, and Claude Desktop.

## Features

- **Full GeoServer Management**: Manage workspaces, stores, layers, layer groups, styles, users, and system configuration via MCP tools
- **Multi-Transport Support**: Supports Streamable HTTP and SSE remote transports for different MCP clients
- **One-Click Deployment**: Containerized deployment via Docker with pre-configured environment variables, ready to use
