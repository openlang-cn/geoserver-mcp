# Code of Conduct

## Our pledge

We want this project to be an open, respectful, and technically constructive community.
Contributors, maintainers, and users are expected to participate in ways that make collaboration safe and productive for everyone.

## Expected behavior

- Be respectful and specific in technical discussions.
- Assume good intent, but challenge ideas with evidence.
- Focus feedback on code, design, and behavior, not on individuals.
- Help keep issue and pull request discussions on topic.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language.
- Bad-faith disruption, intimidation, or deliberate hostility.
- Publishing private information without permission.
- Repeated off-topic or abusive communication.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, and contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainers through the repository owner or a private GitHub security/advisory channel when available.
